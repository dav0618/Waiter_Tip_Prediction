1. The aim of the project.

   The aim of this project is to predict most appropriate tip for waiter service.

2. How test the code.

   It's very simple, just download data and you can run it on Jupyter Notebook, or in Google colab.
   
3. What libraries are used.

   Pandas, MatPlotLib and Sklearn libraries for working with data, visuilising some relationships of data, and of course to get model.  

4. Results of the model.

   The accuracy of model is 44%.
